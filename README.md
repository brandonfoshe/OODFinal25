# OODFinal25
Calendar Final for Object Oriented Design 2025
## Design Patterns Impleneted
Command is used for interactions with the Firebase and Events, Such as Creation of an event, Editing of an event, and Deletion of an Event
Observer is used to demonstrate a Notification system for a simulated Week, I was unable to fully understand emailJS at the moment i might get to it tomorrow
Factory is used for the creation of the event object
Builder is used for the setting of the event object's variables and a returned Objected WorkEvent or PersonalEvent
##
Features
##
Creation of Events
  -Setting Event Time
  -Setting Event Type : Personal or Work
  -Setting of Duration in Minutes
  -Setting of Description
Editing of Events
  -Edit of any variable of Event
Deltion of Events
  -Deletion of Events from the Calender
Indivdual Displays of Events given a Day of the Week
A large Week-Long Display
  -A simulated Week with Notification for the Given Email
